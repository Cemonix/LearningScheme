# LearningScheme

A personal repository for learning Scheme (Guile) from scratch. Contains small programs and exercises implemented while getting familiar with functional programming concepts like recursion, tail calls, and higher-order functions.

## Implemented so far

- Factorial (tail-recursive with accumulator)
- Fibonacci
- `map` reimplementation
- Fourier Epicycles renderer (`epicycles.scm`) — takes a 2D path, runs a DFT, extracts frequency/amplitude/phase components, and renders the reconstructed path as a PPM image
